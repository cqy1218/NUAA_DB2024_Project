## Run

1.使用MySQL运行setup.sql脚本，建立初始数据库系统；

2.运行Python程序main.py，即可进入选课管理系统；该系统具有较好的用户交互界面，参照界面描述进行操作即可。

注：run.sql为实验1至实验5的SQL语言指令脚本，使用的是Oracle软件11g32位版本。
